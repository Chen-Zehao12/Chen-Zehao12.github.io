ANN 基础 (教程第 3、4、5 章）

1.ANN 的要素：结构，激活函数，参数；写出 ANN 的解析式并就此说明 ANN 的表达能力；分析 ANN 能力的影响因素。

  • 结构：PPT1、2，P32
  • ANN 三要素：传递函数（激活函数）、拓扑结构、阈值和权值

  • 人工神经网络：可以看作是一个信息处理模块，其功能就是实现一种变量或特征形式到另一种变量或特征形式的映射。

  • 激活函数

    • sigmoid 函数
    • tanh 函数
    • ReLU 函数
    • linear 函数
    • softmax 函数

  • 决定 ANN 功能和应用的三要素

    • 神经元激活函数
    • 网络拓扑结构
    • 结构权值学习算法

2.BP 算法推导；分析说明梯度消失或爆炸的原因；分析说明参数惩罚的意义

  • BP 算法推导：PPT3，P35
  • 参数惩罚的意义：限制模型的能力，从而降低过拟合风险

3. 算法实例推导：写出具体的表达式，探讨跃层结构意义。

4.DNN 的优化策略：问题与策略及推导（梯度消失与爆炸，局部优化，过拟合）（参数惩罚（参数共享与绑定，了解 L0,L1,L2 惩罚的不同影响），SGD 与 minibatch，BN，Adam，早停，dropout）；

  • 《深度学习实战（教程）》：第 4 章
  • L0 惩罚：我们将参数不等于 0 的个数控制在 c 以内来达到限制模型的目的
  • L1 惩罚：不要求参数的非零数量被限制在某个范围内，但要求参数数值的总和要小于某个数值
  • L2 惩罚：将对参数的绝对值求和改成对参数的平方求和
  • 相比于权重衰减，L1 正则化的限制更为严格，因此也就更加的稀疏（sparse），稀疏性也就是我们最终优化到的参数中有许多 0。稀疏性的一大好处是有利于特征选择（Feature Selection），由于大多数参数都为 0，而这些 0 参数对应的特征就没有被使用，实际上我们就选择出了一些重要特征对数据进行预测，并自动筛选掉一些无用的特征，这些无用特征很大程度上也有可能是噪声特征。
  • SGD（随机梯度下降）：《深度学习实战（教程）》：P61
  • minibatch：《深度学习实战（教程）》：P62
  • BN：《深度学习实战（教程）》：P249
  • Adam：《深度学习实战（教程）》：P244
  • 早停：《深度学习实战（教程）》：P189
    • 我们就期望能在验证错误率的最低点（或最低点附近）停止训练算法
  • dropout：《深度学习实战（教程）》：P193

6.Char4 实例：dropout 实现；参数初始化策略

7.Char5 实例：Adam 算法实现；BN 算法推导与实现
