3.1 定义和初始化

  • vector v1(v); // 定义一个 vector 副本
  • vector v2(n, i); // 定义了长度为 n 的 vector v2，并且每个元素都是 i

3.2 插入元素

在 vector 中插入元素包括两种，一种是在尾部添加元素，使用的函数是 push_back() 函数，另一种是在指定位置插入元素，使用到的函数是 insert() 函数。

其中，insert() 函数的具体形式为：insert(p, t)，指的是在迭代器 p 之前插入值为 t 的元素。
