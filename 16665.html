map 基本操作

发表于2020-10-06|更新于2022-01-21|C++-STL库
|字数总计:710|阅读时长:3分钟

1. map 简介

标准库 map 类型是一种以键 - 值 (key-value) 存储的数据类型。

2. map 头文件

在 C++ 中使用 map 需要包含头文件

3. 基本操作

3.1 定义和初始化

map 是键 - 值对的组合，有以下的一些定义的方法：

  • map<k, v> m; // 定义了一个名为 m 的空的 map 对象
  • map<k, v> m(m2); // 创键了 m2 的副本 m
  • map<k, v> m(b, e); // 创建了 map 对象 m，并且存储迭代器 b 和 e 范围内的所有元素的副本

3.2 插入元素

在 map 中元素有两种插入方法：

  • 使用下标

    • 在 map 中使用下标访问不存在的元素将导致在 map 容器中添加一个新的元素

  • 使用 insert()函数

    • map_name.insert(e)
    • map_name.insert(beg, end)
    • map_name.insert(iter, e)

    上述的 e 为一个 value_type 类型的值。beg 和 end 标记的是迭代器的开始和结束。

3.3 查找元素

若只是查找该元素是否存在，可以使用函数count(k)，该函数返回的是 k 出现的次数；

若是想取得 key 对应的值，可以使用函数find(k)，该函数返回的是指向该元素的迭代器。

3.4 删除元素

从 map 中删除元素的函数是 erase()，该函数有如下的三种形式：

  • map_name.erase(k) // 删除 m 中键为 k 的元素，返回删除的元素的个数
  • map_name.erase§ // 删除的是迭代器 p 指向的元素，返回 void
  • map_name.erase(b, e) // 删除的是迭代器 b 和迭代器 e 范围内的元素，返回 void
