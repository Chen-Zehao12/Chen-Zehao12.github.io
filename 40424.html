1. Matplotlib 介绍

​ Matplotlib 是一个 Python 的 2D 绘图库，它以各种硬拷贝格式和跨平台的交互式环境生成出版质量级别的图形。通过 Matplotlib，开发者可以仅需要几行代码，便可以生成绘图。一般可绘制折线图、散点图、柱状图、饼图、直方图、子图等等。Matplot 使用 Numpy 进行数组运算，并调用一系列其他的 Python 库来实现硬件交互。

​ matplotlib.pyplot 是一个有命令风格的函数集合，它看起来和 MATLAB 很相似。每一个 pyplot 函数都使一副图像做出些许改变，例如创建一幅图，在图中创建一个绘图区域，在绘图区域中添加一条线等等。在 matplotlib.pyplot 中，各种状态通过函数调用保存起来，以便于可以随时跟踪像当前图像和绘图区域这样的东西。

4. 常用函数

函数 说明
plt.figure() 创建一个新的画布，或激活一个现有的画布
plt.plot() 可以绘制点和线、折线图、曲线图
plt.show() 显示所有已打开的画布
plt.subplot() 添加子图
plt.savefig() 保存当前图形，注意：该函数一定要放在 plt.show() 之前
fig.add_subplot() 使用 ax 的方式添加子图，操作比 plt.subplot() 更加灵活
plt.rcParams[“font.family”]=“SimHei” 解决中文显示问题，SimHei 为中文简体黑体
ax.set_xlim() 设置 x 轴的刻度范围
ax.set_ylim() 设置 y 轴的刻度范围
ax.set_xticks() 设置 x 轴的刻度
ax.set_yticks() 设置 y 轴的刻度
ax.set_xticklabels() 设置 x 轴的刻度标签
ax.set_yticklabels() 设置 y 轴的刻度标签
ax.tick_params() 装饰轴上的刻度线和轴标签
ax.set_xlabel() 设置 x 轴坐标标签
ax.set_ylabel() 设置 y 轴坐标标签
ax.set_title() 设置图的标题
ax.legend() 添加图例
ax.text() 在图中指定位置添加文本
ax.annotate() 在图文指定位置添加箭头和文本
ax.spines() 装饰坐标轴
ax.bar() 绘制垂直条形图
ax.barh() 绘制水平条形图
ax.scatter() 绘制散点图
ax.boxplot() 绘制箱线图
ax.pie() 绘制饼图
plt.grid() 添加网格
