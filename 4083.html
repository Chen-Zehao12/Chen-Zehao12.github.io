(一)Windows 下安装 Gephi

发表于2020-08-24|更新于2022-01-21|社交网络-Gephi
|字数总计:538|阅读时长:1分钟

1. Java 开发工具包 JDK

Gephi 需要用到 Java 的开发工具包 JDK，因此需要先安装 JDK。如果电脑中已安装并已配置好 JDK，可忽略这一步。

  • 进入 JDK 官方网站下载相关的 JDK 安装工具包

    进入官网 https://www.oracle.com/java/technologies/javase-downloads.html

    选择 JDK Download

    选择 jdk-14.0.2_windows-x64_bin.exe

    进行下载

    下载完成后一直下一步安装完成即可。记住 JDK 的安装目录。

  • 配置环境

    使用鼠标右键点击【此电脑】-【属性】-【高级系统设置】开启系统属性的界面，在【系统属性】的界面中切换至【高级】选项卡，点击打开下面的【环境变量】

    在这里我们新建一个系统变量，如下：点击【新建】，在新建的界面中，设置输入变量名为 JAVA_HOME，变量值指向你安装 JDK 的目录，这里为 C:\Program Files\Java\jdk-14.0.2，完成后点击【确定】

    接下来，点击【新建】 ，弹出窗口，环境的变量：CLASSPATH ，变量值为：.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar

    接下来的在系统变量中找到变量名为 PATH 的变量。选择该变量后点击【编辑】，在编辑的界面中加入下面两个变量路径。

    %JAVA_HOME%\bin

    %JAVA_HOME%\jre\bin

    最后，Win+R 打开运行 Windows，输入 CMD 回车，输入 java --version 的命令，就可以看到你安装 JDK 的版本号，表示安装配置成功。

2. Gephi

进入官网 https://gephi.org/

点击 Download FREE

点击 Download Gephi for Windows 进行下载

下载完成后一直下一步安装完成即可。记住安装目录。

安装完成后，打开程序即可。

如果出现 Cannot find Java 1.8 or higher. 的错误，可以按照下述方法进行解决。

找到 Gephi 的安装路径，修改 etc 下的 gephi.conf 文件（记事本打开），此处为 C:\Program Files\Gephi-0.9.2\etc

去掉 jdkhome 前面的注释符 #，将 jdkhome 路径修改为 JDK 安装路径，此处为 jdkhome=“C:\Program Files\Java\jdk-14.0.2”

保存即可。

然后就可以打开 Gephi 了

于此，Gephi 便安装完成了。
