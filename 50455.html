1. 人工智能编程语言

10 种人工智能 Artificial Intelligence (AI) 和机器学习 Machine Learning (ML) 编程语言

  • Python

  • R-language

    （1）开源，免费开发。

    • ** 开源：** 可以修改源代码

    • ** 免费开发：** 免费提供给用户使用

    • 二者不同

    （2）解释型和动态类型的编程语言

    • ** 解释型：** 类似同声传译

    （3）用于统计分析和控制数据

    （4）多种操作系统运行

  • Java

  • Lisp（List Processing）

    （1）最古老的 AI 编程语言，主要为列表

    （2）第一个函数式程序语言

    • ** 函数式：** 关系的是映射关系

    （3）是解释型和动态类型的编程语言

    （4）有效地处理符号信息

  • JavaScript

    （1）一种开源的轻量级的高级 web 应用程序客户端编程语言

    （2）是解释型和动态类型的编程语言

    （3）数据科学和机器学习（Tensorflow.js）

  • Prolog（Programming in Logic）

    （1）解释型 or 编译型都可以，没有完全的标准化

    （2）逻辑式编程语言

    （3）用来建造专家系统、自然语言理解、智能知识库

  • Haskell

    （1）函数式编程语言

    （2）静态类型编程语言

    （3）支持嵌入式领域特定的语言

  • Julia

    （1）函数式编程语言

    （2）编译型和动态类型语言

    （3）计算科学和数值分析，机器学习

  • C++

  • AIML（Artificial Intelligence Markup Language）

    （1）人工智能标记语言

    （2）人工智能和机器学习

2. Python 概述

2.1 优势

（1）脚本语言 + 语句执行

  • ** 脚本语言：** 调用的时候解释和编译，是解释型语言

（2）简洁 + 强制可读性

（3）跨平台 + 开源

（4）面向过程 + 面向对象（Java 强制面向对象）

（5）图形界面

缺点：

（1）代码不能加密

（2）相较于 C/C++ 速度不如

2.2 应用范围

（1）操作系统管理、服务器运维的自动化脚本

（2）科学计算

（3）Web 应用

（4）图形用户界面开发

（5）其他：游戏开发

（6）……
